# 三すくみバトル（5人抜き）

このアプリは、ドラクエ風の簡易ターン制バトルゲームです。プレイヤーは「斬/突/打」から行動を選び、5人の敵を連戦で倒すことを目指します。

## 必要環境
- Java ランタイム: JDK/JRE 25 以上を推奨
- 画面サイズ: 640x480 以上

## 起動方法
- Windows: `bin/tri-battle-gauntlet.bat` をダブルクリック、または実行
- macOS/Linux: ターミナルで `bin/tri-battle-gauntlet` を実行（必要なら `chmod +x`）

配布ZIPを展開すると `bin/` と `lib/` フォルダが含まれています。`bin/` のスクリプトから起動してください。

## ゲームの遊び方
1. タイトルで「開始」を選びます。
2. ステータス配分画面で 力/技/守備 に合計18ポイントを割り振り、「このステータスで開始」。
3. 戦闘では「斬/突/打」から1つを選びます。
   - 相性: 斬>打、打>突、突>斬（有利で命中+20%、ダメージ+5／不利で命中-20%、ダメージ-5）
   - 命中率: 70% + 技×2%（5〜95%にクランプ）
   - ダメージ: 10 + 力×3 − 守備×2（最小1）
4. 5人抜き: 先鋒→次鋒→中堅→副将→大将の順に連戦。勝利ごとにHPが20回復（上限100）。
5. 途中でHPが0になるとゲームオーバー。5人すべて倒すと完全勝利です。

## 操作
- マウス: ボタンをクリック
- キーボード: 矢印キーで選択、Enterで決定（対応している画面のみ）
- Esc: タイトルに戻る（確認ダイアログあり）

## トラブルシューティング
- 起動しない/真っ白: JDK/JRE 25 が導入されているか確認してください。
- 文字化け: UTF-8対応の環境/フォントを使用してください。
- セキュリティ警告（macOS/Linux）: 実行権限付与 `chmod +x bin/tri-battle-gauntlet` の後に実行してください。
